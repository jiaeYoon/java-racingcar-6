## 자동차 경주 기능 명세
2. 경주할 자동차 객체 생성
   1. 자동차 이름 입력
      1. 쉼표 기준 split
      2. 검증 : 5자 이하 문자
   2. 이름을 가지는 자동차 객체 생성
3. 시도 횟수 입력
   1. 검증 : 숫자 
4. 자동차 전진/멈춤
   1. 무작위 값 생성
   2. 4 이상이면 전진
5. 차수별 실행 결과 출력
6. 최종 우승자 출력
   1. 이동거리가 가장 큰 자동차 도출
   2. 자동차가 2개 이상이면 쉼표로 구분

자동차 - 거리, 이름
    이름 - 쉼표 기준 구분, 5자 이하
    전진 조건
이동횟수(turn) 입력
util - 무작위 값, split
결과 - 우승자 한 명 이상인 경우 쉼표로 구분
